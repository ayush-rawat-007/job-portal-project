💼 Job Portal Project
📌 Description

A full-stack Job Portal web application that allows users to search and apply for jobs while enabling recruiters to post and manage job listings.

🚀 Features
User Registration & Login
Job Search Functionality
Apply for Jobs
Recruiter Job Posting
Dashboard for managing applications
🛠️ Tech Stack
Frontend: (HTML, CSS, JS / React)
Backend: (Python / Django)
Database: (SqlLite)


⚙️ How to Run
Clone the repository
Install dependencies
Run the server
📊 Future Improvements
Add filters for job search
Improve UI/UX
Add email notifications
